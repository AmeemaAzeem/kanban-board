# Kanban Board Web Application

A **dynamic, interactive Kanban board** built with **HTML, CSS, and JavaScript**, inspired by **Notion Light Mode** for a clean, modern, and vibrant interface. This project demonstrates front-end development skills, including **DOM manipulation, drag-and-drop functionality, modal inputs, responsive design, and local storage persistence**.

## Features

* **Add, Edit, Delete Tasks:** Manage your tasks with a professional modal input.
* **Drag & Drop:** Move tasks between columns seamlessly.
* **Priority Tags:** Assign Low, Medium, or High priority to tasks, visually indicated by colored dots.
* **Persistent Storage:** Tasks and their statuses are saved in **LocalStorage**, ensuring data is retained after page refresh.
* **Responsive & Aesthetic UI:** Designed with vibrant colors (#FE5E9E background, #FF8348 task cards, #FDAD66 shadows) and smooth hover animations for an engaging user experience.
* **Dimension & Shadow Effects:** Task cards and columns have subtle shadows and hover scaling for a 3D feel.

## Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Storage:** Browser LocalStorage
* **Design Inspiration:** Notion Light Mode
